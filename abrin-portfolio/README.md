# Abrin's Portfolio

This project is a personal portfolio website for Abrin Rithish G S, showcasing skills and projects as a Machine Learning Engineer.

## Project Structure

- **abrin.html**: The main HTML file containing the structure of the portfolio, including sections for hero, about, skills, projects, education, and contact information. It also includes an embedded map, image hotspots, and styled anchor links.

- **style.css**: The stylesheet for the portfolio, which includes external, internal, and inline CSS. It defines the visual appearance, typography, layout, and hover effects.

- **Abrin.pdf**: The resume of Abrin Rithish G S, available for download through a button on the portfolio website.

## Features

- Responsive design with a focus on a professional appearance.
- Use of modern CSS techniques such as glassmorphism and layered gradients.
- Interactive elements using CSS-only hover effects.

## How to Navigate

- Click on the "Download Resume" button to access the PDF version of the resume.
- Explore the skills and projects sections to learn more about Abrin's expertise and work.
- Use the contact information to connect via phone or social media links.

## Technologies Used

- HTML
- CSS
- No JavaScript or frameworks used.
